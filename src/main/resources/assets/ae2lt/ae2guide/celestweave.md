# Celestweave

Celestweave is a modular high-end armor set. Install an Ultimate Overload Core and compatible modules at the Overload Device Workbench, then equip the set to power movement, protection, and utility features from its stored FE and bound ME network.

Each piece has its own module slots. The Device Hub shows installed modules, energy, network status, and runtime toggles.
